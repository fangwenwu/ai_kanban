我需要开发一个【股票涨跌/横盘智能判断看板】页面，用于直观展示单只股票当前趋势状态（上涨/下跌/横盘），并展示核心判断依据数据与可视化图表。

一、看板核心功能

1. 顶部展示股票当前趋势结论：明确标注【上涨趋势 / 下跌趋势 / 横盘震荡】
2. 趋势判断依据模块：展示核心指标状态，用颜色区分（红=涨，绿=跌，灰=横盘）
3. 可视化图表模块：展示K线、均线、成交量、MACD、BOLL、RSI等指标图表
4. 特征总结模块：文字总结当前股票的上涨/下跌/横盘关键特征

二、必须展示的核心指标与数据

1. 趋势指标
   - 均线排列（5/10/20/60日）：多头排列/空头排列/粘合缠绕
   - MACD：金叉/死叉/零轴位置
   - BOLL：上轨/中轨/下轨位置，开口/收口状态
   - RSI：数值与趋势区间（>50涨，<50跌，40-60横盘）

2. 量能数据
   - 当日成交量、量比、是否放量/缩量
   - 量价关系：价涨量增/价跌量增/无量震荡

3. K线形态
   - 当前K线类型（大阳/大阴/小阴小阳/十字星）
   - 趋势形态（突破/破位/横盘）

4. 资金数据
   - 主力资金净流入/净流出
   - 换手率

三、横盘震荡专属判断规则（必须集成）

1. BOLL带收口变窄
2. 均线粘合缠绕
3. 成交量持续萎缩
4. 股价波动幅度<5%
5. RSI在40-60之间
   满足3项以上判定为横盘

四、UI风格要求

1. 简约金融看板风格，深色/浅色双主题
2. 趋势结论大字体突出显示
3. 指标用色块标签展示：上涨(红)、下跌(绿)、横盘(灰/蓝)
4. 图表可交互，支持悬停查看数据
5. 适配PC端，布局清晰直观

五、输出要求
输出可直接开发的前端页面方案，包含：页面结构、模块划分、展示字段、颜色规则、图表配置、判断逻辑规则。
